# AutoParts.Client

Front-end per projektin final te shkolles. Ky projekt perdor HTML, Bootstrap dhe JavaScript te thjeshte.

## Si ta hapesh

1. Hape back-end-in fillimisht.
2. Hape kete folder ne VS Code.
3. Hape `index.html` me Live Server.

Front-end pret qe API te punoje ketu:

```js
http://localhost:5000/ArtikujtAuto
```

Nese API perdor nje port tjeter, ndrysho `API_URL` brenda `scripts/store-http.js`.
